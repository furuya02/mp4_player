# mp4_player

カレントディレクトリのMP4ファイルを順次再生するシンプルな動画プレーヤーです。

## 概要

`mp4_player` は Python（OpenCV + tkinter）で構築された軽量な動画プレーヤーです。カレントディレクトリの `.mp4` ファイルをスキャンし、ダークテーマのGUIで順次再生します。

## 機能

- カレントディレクトリの全MP4ファイルを順次再生
- ダークテーマの映像表示GUI
- 再生 / 一時停止 / 停止
- 次へ / 前へ ナビゲーション
- シークバーで任意の位置にジャンプ
- ファイル一覧パネル（クリックで選択再生）
- 再生終了時に自動的に次の動画へ
- キーボードショートカット

## インストール

### pip を使用

```bash
git clone https://github.com/furuya02/mp4_player.git
cd mp4_player
pip install -e .
```

インストール後、`mp4_player` コマンドが使えるようになります：

```bash
mp4_player
```

## 使い方

MP4ファイルがあるディレクトリに移動して実行します：

```bash
cd /path/to/your/videos
mp4_player
```

### キーボードショートカット

| キー | 操作 |
|------|------|
| `Space` | 再生 / 一時停止 |
| `→` | 次の動画 |
| `←` | 前の動画 |
| `Esc` | 終了 |

### ボタン操作

| ボタン | 操作 |
|--------|------|
| ⏮ 前へ | 前の動画 |
| ▶ 再生 / ⏸ 一時停止 | 再生 / 一時停止 |
| ⏹ 停止 | 停止（先頭に戻る） |
| ⏭ 次へ | 次の動画 |

## 動作要件

- Python 3.10 以上
- tkinter（通常 Python に同梱）

以下の依存パッケージは自動的にインストールされます：
- opencv-python >= 4.8.0
- Pillow >= 10.0.0

> **注意:** このプレーヤーは映像のみの再生です（音声再生は非対応）。音声対応は将来のリリースで追加予定です。
